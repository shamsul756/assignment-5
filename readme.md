1️⃣ What is the difference between var, let, and const?
ans:var : It is the old way to declare variables.It is function scoped.
The value can be changed and redeclared.

let : Introduced in ES6 (modern JavaScript).It is block scoped .The value can be changed, but cannot be redeclared in the same scope.

 const : Also introduced in ES6.It is block scoped.The value cannot be changed or redeclared. 
 2️⃣ What is the spread operator (...)?

 ans: In JavaScript, the spread operator (...) is used to expand or copy elements of an array or object. It helps to combine arrays, copy arrays, or add new values easily.

3️⃣ What is the difference between map(), filter(), and forEach()?
 ans: map() : Used to create a new array. It transforms each element of the array. Always returns a new array.

filter() :Used to select specific elements from an array.It returns a new array with elements that match the condition
